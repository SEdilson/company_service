# Company Service

### Running

User pipenv shell:
```
    pipenv shell
```

Run dependencies with docker-compose:
```
    docker-compose up -d
```

Run the application:
```
    python3 manage.py runserver
```

### Running Tests

Run dependencies with docker-compose:
```
    docker-compose up -d
```

Run the tests:
```
    python3 manage.py test
```
